# 图片预览页

1. 默认图片适应屏幕大小展示
2. 预览图片
    - 单击图片进入准沉浸模式，背景变为黑色，隐藏所有icon 按键，再次点击图片退出沉浸模式，
    - 默认右滑上一张、左滑下一张、双击放大到100%原始图片大小、再次双击撑满屏幕
    - 支持双指缩放、双指旋转
    - 当图片被放大时，上下左右滑动只是移动图片，当滑动到当前图片的边界时，要停止移动，继续向左或向右滑动才会进入下一张或上一张。
3. 顶部控件
    - 点击返回按钮：返回上一页，关闭当前预览页。
    - 显示文件名，文件名下方是图片创建时间，格式是：YYYY-MM-DD HH:mm:ss。时间右侧是图片格式、标签。
    - 右侧是沉浸式查看 icon 按钮，点击后进入沉浸模式，背景变为黑色，隐藏所有icon 按键，再次点击 icon 退出沉浸模式。
        - 进入沉浸模式时，底部会变成两个 icon 按钮：上一张，下一张，单击一下就是切换一张，长按 2s 就是连续切换对应的上一张或下一张图片，图片切换的模式是直接即刻切换，无需滑动动效或者渐隐渐显的动态效果。
4. 底部控件
    - 点击添加到相册：打开相册列表弹窗，选择相册或新建相册，将图片添加进去。
    - 点击添加标签：打开添加标签弹窗，选择或新建标签，为图片添加标签，方便后续筛选。
    - 点击评分：为图片评分，评分范围是 0-5 分。
    - 点击删除按钮：选择永久删除、删除到回收站（90 天内可恢复）、移出当前相册（只是从当前相册移除，不删除图片，成为一个未分类的图片，应该归类为“未分类”相册）。
    - 点击更多按钮：显示更多操作选项
        - 分享：将图片分享到其他应用。
        - 设置为壁纸：将图片设置为壁纸。
        - 详情：查看图片的详细信息，如文件大小、文件尺寸、创建时间、修改时间、标签等。
        - 重命名：输入新的图片名称，保存后，图片的文件名会更新。
        - 添加描述：输入图片的描述，保存后，图片的描述会更新。

# 照片页
1. 照片区
    - 默认按照添加时间排序显示图片，2、3、4、5、列时，按天为单位排序，8、16 列时按月为单位排序
    - 当滑动页面时，右侧边缘会显示一个滑块时间轴，点击滑块可以唤起时间轴选择器，选择后会快速跳转到对应的时间下，也可以手动拖动滑块进行滑动。
    - 支持双指缩放不同布局下的图片预览图大小，默认采用瀑布流布局。具体详见布局切换介绍。
    - 支持长按触发多选，启动多选时，左上角变为退出多选按钮，点击后退出多选状态。右上角变为选择方式按钮，点击后切换按日选择或按月选择。
        - 选择按日选择时，图片按天为单位排序，点击日期右端的全选按钮即可全选该天的所有图片。
        - 选择按月选择时，图片按月为单位排序，点击日期右端的全选按钮即可全选该月的所有图片。
        - 点击退出多选按钮，会退出多选状态，同时会取消所有选中的图片。
2. 顶部操作按钮：
    - 搜索：点击搜索按钮，输入图片名称、标签，搜索符合条件的图片。
    - 布局切换：瀑布流、自适应、宫格、列表四种。
        - 瀑布流：默认是瀑布流布局，图片按照相等的宽度，自动调整高度，填充屏幕，流式布局，双指可缩放，支持 2、3、4、5、8、16列布局。
        - 自适应：图片按照相等的高度，自动调整宽度，填充屏幕，双指可缩放，支持 最低60px，最高100%的高度，自动调整宽度和横向的图片数量。
        - 宫格：图片以正方形宫格形式展示，双指可缩放，支持 2、3、4、5、8、16列布局。
        - 列表：图片以列表形式展示，列表有左边缩略图、文件名+扩展名、文件大小、文件尺寸、添加时间。不可缩放。
    - 排序方式：
        - 添加时间、创建时间、修改时间、文件名、文件大小、文件尺寸、评分
        - 每项排序方式都支持正序倒叙
    - 筛选：支持多选交叉筛选
        - 图片名称、扩展名、标签、评分、文件尺寸（横向、纵向、方形、选择范围）、文件大小（弹窗输入范围）、添加时间（弹窗选择范围）、修改时间（弹窗选择范围）、创建时间（弹窗输入范围）、
        - 筛选方式：包含、不包含
    - 更多：
        - 最近删除：进入回收站，查看最近删除的图片。
        - 释放空间：点击进入释放空间界面
            - 清理截屏（待定）
            - 清理相似图片（待定）
            - 清理重复内容（待定）
            - 清理缓存（待定）
        - 设置：点击设置按钮，进入设置页，调整相册的显示设置。
    - 进入多选时，顶部操作按钮会变成：
        - 左侧角退出多选：取消多选状态。
        - 右侧角全部选择：选择所有图片（待定，可能造成性能问题）。
3. 底部悬浮导航栏分为两个 tab 分别对应照片、相册，默认是照片icon高亮显示
    - 点击照片icon：进入照片页，显示所有图片。
    - 点击相册icon：进入相册页，显示所有相册。
    - 进入多选是隐藏 tab，变为：
        - 添加到相册：打开相册列表弹窗，选择相册或新建相册，将图片添加进去。
        - 添加标签：打开添加标签弹窗，选择或新建标签，为图片添加标签，方便后续筛选。
        - 评分：为图片评分，评分范围是 0-5 分。
        - 删除：选择永久删除、删除到回收站（90 天内可恢复）、移出当前相册（只是从当前相册移除，不删除图片，成为一个未分类的图片，应该归类为“未分类”相册）。
        - 分享：将图片分享到其他应用。
# 相册页
1. 相册区从上到下依次是：常用相册、自定义相册、回收站，常用相册和自定义相册采用宫格布局展示，横向三个相册，每个相册显示封面图、相册名称、图片数量。
    - 常用相册区：
        - 默认展示：全部照片、相机、截屏录屏、视频
        - 点击更多：显示更多自动相册：
            - 标签相册：根据图片的标签自动分类图片。
            - 评分相册：根据图片的评分自动分类图片。
            - 扩展名相册：根据图片的扩展名自动分类图片。
            - 文件尺寸相册：根据图片的文件尺寸自动分类图片，默认按照横向、纵向、方形的图片尺寸分类，也可以选择范围。
            - 文件大小相册：根据图片的文件大小自动分类图片，默认按照 0-10、10-50、50 分类，也可以选择范围。
    - 自定义相册区：即为用户自己创建的自定义相册。不包括常用相册。
        - 默认展示：6 个置顶的自定义相册
        - 点击更多：显示所有的自定义相册。
    - 回收站区：简单一个横向撑满的列表按钮，点击后进入回收站，查看最近删除的图片。
2. 定部操作按钮：
    - 搜索：点击搜索按钮，输入图片名称、标签，搜索符合条件的图片。
    - 更多：
        - 最近删除：进入回收站，查看最近删除的图片。
        - 释放空间：点击进入释放空间界面
            - 清理截屏（待定）
            - 清理相似图片（待定）
            - 清理重复内容（待定）
            - 清理缓存（待定）
        - 设置：点击设置按钮，进入设置页，调整相册的显示设置。
